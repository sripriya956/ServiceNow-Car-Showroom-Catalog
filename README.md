# ServiceNow Car Showroom Service Catalog Project

## Project Overview
This project implements a Car Showroom Service Catalog in ServiceNow.

It includes:
- Catalog Creation (Mahendra)
- Categories (Sudan, XUV, Sports)
- Catalog Items (Polo, Thar, XUV700)
- User, Role, Group Creation
- Custom Table (Cars Fulfillment)
- Workflow with Multi-level Approval
- Task Creation
- Email Notifications
- Service Portal Testing
